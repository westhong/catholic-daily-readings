# Email Draft 2 — 給 FHL 信望愛信仰與聖經資源中心

**收件人：** bible@fhl.net
**主題：** 關於使用思高譯本聖經經文用於天主教每日讀經項目 — 私人研究用途

---

敬愛的 FHL 信望愛團隊，

首先，感謝你們多年來提供如此豐富的聖經資源給華人教會。信望愛的聖經閱讀工具陪伴了無數信徒的每日靈修生活，我本人也從中獲益良多。

**關於我的項目：**

我叫 West，現居加拿大，是一名獨立開發者。我正在建立一個名為「Catholic Daily Readings」的开源项目（GitHub: https://github.com/westhong/catholic-daily-readings），旨在為全球華人天主教徒提供完全免費的每日彌撒讀經服務。

目前項目處於 MVP 階段，已上線運行。本項目沒有任何商業營利目的，不投放廣告，不收取任何費用，完全是個人對神的奉獻。

**現況說明：**

目前我使用 FHL 提供的 API（bible.fhl.net）來獲取思高譯本聖經經文，用戶運行腳本後，經文是即時 fetch 並顯示給用戶的，並非預先下載儲存。

我已在 GitHub 和項目說明中清楚標明：
- 所有經文版權歸香港聖經公會 / 信望愛資源中心所有
- 僅供私人研究使用
- 附有適當的版權聲明

**我希望達到的目標：**

我相信天主的聖言應該被自由地傳播給所有人。華人天主教社群目前沒有任何免費的每日讀經系統，這是我的項目存在的意義。

我也理解思高譯本有相應的版權保護，因此我想請問：

1. **私人研究用途的範圍**：我目前標示的「僅供私人研究使用」是否足夠？還是需要更正式的授權機制？

2. **開放許可的可能性**：是否有可能將本項目使用的聖經經文部分，採用 Creative Commons 或類似的開放授權，讓全球華人天主教徒能夠更方便地獲得每日的靈修材料？

3. **联系方式**：如果需要進一步討論授權事宜，請問是否有專門負責版權或合作事宜的窗口？

**我可以做到的：**

- 所有經文顯示時，完整保留版權聲明
- 明確標示經文來源為信望愛 / 香港聖經公會
- 如果需要任何費用或授權條款，願意商討

這個項目是為了榮耀天主、服務神的教會。我期待聽到您的回覆。

願天主祝福你們的服侍。
