# 版權來往記錄

> 更新日期：2026-04-21

本頁記錄 Catholic Daily Readings 項目與各地聖經版權持有機構的來往通訊，供公眾查閱，亦作為本專案遵守版權法的公開承諾。

---

## 核心原則

**無需許可即可使用的內容：**
- 讀經引用（章:節，例如「路加福音 24:13-35」）
- 節日名稱（復活期第三主日、聖母升天節等）
- Lectionary Number（禮儀年讀經編號）

**需要版權授權的內容：**
- 聖經譯文的具體文字（譯者的遣詞造句）

> 依據：USCCB 版權部門 Mary Sperry 明確回覆：「No permission is needed for using the reading citations and the names of the various feast days.」

## 授權目標

本專案採用 MIT 開源授權，所有版權來往均為公益用途。我們的目標：

1. **中文讀者**：提供思高譯本（已過保護期，已聯繫確認）
2. **英語讀者**：提供當地認可的聖經譯本（加拿大用戶需聯繫 CCCB）
3. **完全透明**：所有版權來往均公開於此頁

## 貢獻者須知

若您貢獻本專案，請注意：

- **禁止** 直接複製有版權的聖經譯文到程式碼或數據文件中
- 所有經文來源必須來自已獲授權的 API 或公共領域譯本
- 詳細的版權狀態請參考本頁，並在貢獻前提出疑問
